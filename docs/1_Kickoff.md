### Voeg bovenaan **`app.py`** de standaard logging‚Äëconfig toe.

```python
import logging

logging.basicConfig(
    level=logging.INFO,                    # alles ‚â• INFO‚Äëniveau tonen
    format="%(asctime)s [%(levelname)s] %(name)s: %(message)s",
    datefmt="%Y-%m-%d %H:%M:%S",         # Europees datum‚Äëformaat
    force=True,                           # herconfigureer bij auto‚Äëreload
)

```

2. **Waarom?**

   - `logging.basicConfig()` initialiseert √©√©n globale logger‚Äëconfig voor de h√©le app.
   - Je krijgt timestamps, levels en module‚Äënamen out‚Äëof‚Äëthe‚Äëbox.
   - `logger.info()` **flusht** automatisch bij newline‚Äëbuffers, terwijl `print()` op een RPi vaak gebufferd wordt; zo zie je logs meteen wanneer er iets misgaat.
   - Je kunt later per module een eigen logger maken met `logging.getLogger("mijnmodule")`, maar ze volgen dezelfde configuratie.

### Lifespan‚Äëmanager

1. **GPIO¬†setup** (vlak v√≥√≥r de `yield` in de bestaande `lifespan_manager`):

   ```python
   GPIO.setmode(GPIO.BCM)
   GPIO.setup(...)
   #...
   logger.info("GPIO initialised")
   ```

2. **Thread starten** _na_ de setup:

   ```python
   threading.Thread(
       target=gpio_keep_alive,
       daemon=True               # üîë daemon‚Äëthreads stoppen automatisch bij app‚Äëexit
   ).start()
   ```

   threading.Thread voor sync code (zoals RPi.GPIO)

   _`daemon=True`_ betekent dat deze achtergrondthread de applicatie **niet** tegenhoudt om af te sluiten. Zodra het hoofd‚Äëproces eindigt, wordt de daemon‚Äëthread hard gestopt ‚Äì ideaal voor housekeeping‚Äëthreads zoals onze GPIO‚Äëwatcher.

3. **Cleanup** (net v√≥√≥r FastAPI afsluit):

Bij `yield` gaan we over naar de FastAPI app, als we de server stoppen, gaat de code verder na dit punt

```python
# Geef controle aan FastAPI/Socket.IO
yield

# TODO: GPIO cleanup and goodbye
# ...
logger.info("GPIO cleaned up ‚Äì bye!")
```

    """Registreert de knop‚Äëcallback en houdt de thread ‚Äòalive‚Äô."""

    def button_pressed(channel):
        global led_state

        # Doe GPIO en DB dingen

        # ‚¨áÔ∏è¬†We zitten nu in een *andere* thread dan de FastAPI‚Äëevent‚Äëloop
        future = asyncio.run_coroutine_threadsafe(

        logger.debug("Emit scheduled from GPIO thread ‚Äì future: %s", future)
